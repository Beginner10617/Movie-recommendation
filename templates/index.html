Movie Recommendation System

Welcome to the Movie Recommendation System! This web app demonstrates a collaborative filtering-based recommendation engine using two models:

  • Cosine Similarity – based on user ratings similarity
  • Matrix Factorization (SVD) – using latent user and item features

The system also includes interactive data visualizations to help understand the dataset, including rating distributions, similarity heatmaps, and PCA embeddings.
